# Money-Mover

## **Project Goal**
We want to create a budgeting app to help people keep better track of their spending and become more mindful with tailored guidance for their goals and situations.

## **Project Components** 
 - Sign-up page:
   - Create account
   - Set preferences (e.g., saving for a specific goal, decrease spending, etc.)
   - Set the age category to tailor recommendations to where they might be at in life (e.g., high schoolers would mainly only be spending on food or entertainment; adults might pay for transportation, utilities, groceries, etc.)
 - Main Page (post-login)
   - Show an overview of total balance and spendings
   - Breakdowns of specific categories for spending (food, entertainment, rent, utilities, academics, auto, insurance, etc)
   - Visuals (graphs and tables) to show trends
 - AI Assistant Tab
   - Users can get a quick summary of the reports or weekly/monthly statements read out loud to them
   - Users can prompt the assistant to make recommendations for the upcoming month
     - based on transaction history and the goals/preferences set in the user's profile
   - The assistant can make a structured plan for the user tailored to the recommendations
     - could potentially contain some discounts or promos the user could use
 - What-If Tab
   - see how the trends/graphs change if the user were to divert to a different budgeting plan
